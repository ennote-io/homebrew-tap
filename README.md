# Ennote Homebrew Tap

This repository is the official, automated Homebrew Tap for the [Ennote Security CLI](https://ennote.io). 

It is strictly a read-only distribution channel. The Ruby formulas contained in this repository are automatically generated, cryptographically signed, and published by our secure CI/CD pipeline. 

### Installation

To install the Ennote CLI on macOS or Linux via Homebrew, execute:

```bash
brew install ennote-io/tap/ennote
